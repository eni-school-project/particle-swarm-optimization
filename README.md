## Usage

Just launch an either Windows or Linux [release](https://github.com/eni-school-project/particle-swarm-optimization/releases) based on your system and :

- <kbd>Left click</kbd> for adding the target (and changing its place if it already exists)
- <kbd>Right click</kbd> for removing the target
